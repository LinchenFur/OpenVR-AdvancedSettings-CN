# OpenVR高级设置中文版

## 汉化说明
本版本已完成主要界面汉化，包括：
- 主界面
- 音频设置
- 视频设置
- 边界设置
- 接近警告设置

## 编译要求
1. 安装Python 3.7+
2. 配置Python环境变量：
   - 添加Python安装路径到系统PATH
   - 添加Python脚本路径（如`C:\Python37\Scripts`）
3. 重启系统使环境变量生效

## 编译步骤
1. 打开命令提示符
2. 进入项目目录
3. 运行编译命令：
   ```
   build_windows.cmd
   ```

## 安装说明
1. 编译完成后，在`bin\win64`目录找到`OpenVRAdvancedSettings.exe`
2. 运行安装程序
3. 启动SteamVR，插件将自动加载
